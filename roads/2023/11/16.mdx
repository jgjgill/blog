## 신문물 발견!

Road 페이지에서는 숏폼 형식의 글들이 작성되다 보니 글들이 금방 쌓일 것으로 예상된다.

그러다보니 애니메이션이 없으면 많이 심심하게 느껴질 것 같다는 생각을 하게 되었다.

어떻게 할까 고민 좀 했는데 `startViewTransition`을 발견했다!

### 이렇게 편하다니...

정말 간단하게 표현하면 다음 코드가 끝이다!

```ts
document.startViewTransition(() => {
  callback()
})
```

단 몇 줄의 코드만으로 현재 Road 페이지 기능들이 더 부드럽게 느껴지고 재밌어진 것 같다.

빠른게 기술을 적용해봤으니 주말에 깊게 파고들자!
