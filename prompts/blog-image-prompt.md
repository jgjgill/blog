모던하고 미니멀한 흑백 스타일의 기술 블로그 이미지

다음 요소를 반드시 포함할 것:

- 블로그 썸네일 비율 (1200x630, OG 이미지 비율)
- 본문 설명 이미지 (1200 × 675, 16:9)
- 흑백 기반 (black & white only, 색상 사용 금지)
- 단순하지만 의미 있는 구조 (simple but meaningful)
- 텍스트 없이도 주제가 전달되는 직관적인 심볼 중심 구성

디자인 컨셉:

- 단순한 도형이 아니라, 주제를 압축한 개념적 이미지

스타일:

- flat vector 스타일
- 불필요한 디테일 제거 (심플 이즈 베스트)
- 그림자, 그라데이션 사용 금지
- 선 굵기, 정렬, 여백으로 완성도 표현
- 정제된 레이아웃과 균형감 있는 구성

주제 표현:

- 단순한 아이콘 나열이 아닌, 하나의 구조로 메시지를 전달
- 의미가 암시적으로, 매력적으로 드러나도록 구성

전체 분위기:

- 모던하고 세련된 기술 블로그 스타일
- 노션 / 개발 블로그 썸네일 같은 깔끔한 감성
- 과하지 않고, 의도가 느껴지는 미니멀 디자인
