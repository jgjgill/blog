# 어떻게 내가 만든 웹사이트가 구글에 검색되게 할까?🤔

<Callout>
💡 나만의 웹사이트를 만들고 사람들에게 알리고 싶다..! 하지만 어떻게???

이를 위해서는 SEO를 높여야 합니다! 간단하고 쉽게 SEO에 대해서 알아볼까요?!
</Callout>

<br />

## SEO란?
`SEO` 는 검색엔진에 의해 웹페이지의 트래픽 양과 질을 향상시키는 과정입니다. 검색엔진에서 더 찾기 쉽도록 사이트를 개선하는 것이죠.
그리고 비용을 지불하지 않는 트래픽을 목표로 하는 특징이 있습니다.

<br />

## 유용한 404 페이지 만들기

### 404 페이지를 통해 긍정적인 사용자 경험을 유도하세요!

- 잘못된 URL을 입력하면 사이트에 존재하지 않는 페이지로 이동하는 경우 발생
- 이때 사이트에서 제대로 작동하는 페이지로 사용자를 유도
- 사용자가 가장 많이 상호 작용한 웹사이트 게시물을 선택하여 추가 SEO 기회 마련

### 다음은 유용한 맞춤 404페이지를 디자인하기 위한 도움말입니다.

- 요청한 페이지를 찾을 수 없음을 방문자에게 확실히 알립니다. 친근하고 호감이 가는 표현을 사용하세요.
- `404` 페이지의 디자인(탐색 메뉴 포함)이 사이트의 나머지 부분과 같은지 확인합니다.
- 사이트 홈페이지 링크뿐 아니라 가장 인기 있는 기사나 게시물의 링크를 추가합니다.
- 깨진 링크를 신고할 수 있는 경로를 사용자에게 제공합니다.

<br />

## 단순한 URL 활용하기

### 직관적인 URL을 사용해보세요!

암호문과 같은 URL은 사용자에게 익숙하지 않고 혼란을 줄 수 있습니다.

```html
https://www.brandonsbaseballcards.com/folder1/22447478/x2/14032015.html
```

의미 있는 URL은 사용자에게 더 유용하고 다른 문맥에서 쉽게 이해될 수 있습니다.

```html
https://www.brandonsbaseballcards.com/article/ten-rarest-baseball-cards.html
```

URL은 검색결과에 표시됩니다.

일반적으로 URL은 검색결과에서 문서 제목 근처에 표시됩니다.

따라서 직관적이고 간단한 URL을 구성하는 것이 좋습니다.

## 이미지 최적화하기

### 이미지를 통해서도 SEO를 높일 수 있어요!

HTML `<img>` 또는 `<picture>` 요소 사용하기

시맨틱 HTML 마크업을 사용하면 크롤러가 이미지를 찾고 처리합니다. `<picture>` 요소를 사용하면 반응형 이미지를 위한 다양한 화면 크기에 여러 옵션을 지정할 수도 있습니다.

이미지에 `loading=”lazy”` 속성을 사용하면 사용자가 페이지를 더 빠르게 로드할 수도 있습니다.

alt 속성의 내용은 사용자가 스크린 리더와 같은 보조 기술을 사용할 때 그림에 관한 정보를 제공해줍니다. 이와 더불어 가이드에서는 다음과 같이 alt 속성에 대해 추가로 설명하고 있습니다.

> Alt 속성을 사용하는 또 다른 이유는 이미지를 링크로 사용하는 경우 해당 이미지의 대체 텍스트가 텍스트 링크의 앵커 텍스트와 비슷하게 취급된다는 점입니다.

따라서 이미지를 링크로 사용하면 대체 텍스트를 작성하면 Google에서 링크하려는 페이지에 대해 더 잘 이해할 수 있습니다.

이때도 `image1.jpg` `pic.gif` `1.jpg` 와 같은 일반적인 이름은 피하는게 좋겠죠?

## 나아가기

### 사용자의 행동을 분석하세요!

Search Console을 사용해도 사이트가 특별히 대우받는 것은 없습니다.

하지만 문제를 발견하고 해결하는데 도움을 줄 수 있습니다.

해당 서비스에서 웹사이트 소유자는 다음과 같은 작업을 할 수 있습니다.

- 사이트에서 Googlebot이 크롤링하는 데 문제가 있는 부분 확인
- 사이트맵 테스트 및 제출
- robots.txt 파일 분석 또는 생성
- Googlebot이 이미 크롤링한 URL 삭제
- 선호 도메인 지정
- `title` 및 `description` `meta` 태그의 문제 식별
- 사이트에 도달하는 데 사용된 인기 검색어 파악
- Google이 페이지를 보는 방법 확인
- 스팸 정책 위반 통지 수신 및 사이트 재검토 요청

또한, Google 애널리틱스와 같은 웹 분석 프로그램을 통해 사이트에서의 사용자 행동 분석을 할 수 있습니다.

웹 분석 프로그램은 다음과 같이 사용할 수 있습니다.

- 사용자가 내 사이트에 어떻게 찾아오고 어떤 행동을 보이는지와 관련된 정보를 얻습니다.
- 사이트에서 가장 인기 있는 콘텐츠를 확인합니다.
- 사이트 최적화 작업의 영향을 가늠합니다. 예를 들면 `title` 및 `description` `meta` 태그 변경으로 인해 검색엔진에서 유입되는 트래픽이 개선되었나요?
